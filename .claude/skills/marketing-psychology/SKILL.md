# Marketing Psychology — Douglas Elliman Megève

Consulter `.agents/product-marketing-context.md` pour le contexte produit.

## Leviers psychologiques clés pour l'immobilier de luxe

### 1. Rareté (Scarcity)
L'acheteur de luxe est sensible à l'exclusivité, pas aux promotions.
- "Seul bien de ce type actuellement disponible à Megève"
- "Mise en vente confidentielle — disponible sur demande"
- Jamais : "Offre limitée" ou "Prix soldé" — trop grand public

### 2. Perte (Loss Aversion)
La peur de manquer une opportunité est plus forte que le désir de gagner.
- "Ce chalet ne sera plus disponible dans quelques semaines"
- "Plusieurs visites déjà planifiées cette semaine"
- À utiliser avec parcimonie pour garder la crédibilité

### 3. Preuve sociale (Social Proof)
Le luxe se légitime par l'appartenance à un cercle.
- Références à Douglas Elliman comme marque mondiale
- "Nos clients choisissent Megève pour..." + témoignage
- Mentions dans la presse luxe si disponibles

### 4. Autorité (Authority)
Positionner Douglas Elliman comme l'expert indiscutable de Megève.
- Données de marché exclusives
- "Nous avons négocié X transactions à Megève cette année"
- Conseils d'experts dans les posts éducatifs

### 5. Ancrage (Anchoring)
Présenter le prix dans un contexte valorisant.
- Mentionner d'abord les caractéristiques premium, puis le prix
- Comparer à d'autres marchés alpins (Courchevel, Verbier) si le prix est avantageux
- "280 m², vue Mont-Blanc : 3,5 M€" ancre mieux que "Chalet à 3,5 M€"

### 6. Peak-End Rule
L'acheteur se souvient du moment le plus intense et de la fin.
- Ouvrir avec l'élément le plus spectaculaire du bien (vue, spa, localisation)
- Fermer avec une image mentale forte ou un CTA émotionnel
- Pas de liste de features en milieu de légende

### 7. Jobs to Be Done
L'acheteur n'achète pas des m² — il achète une transformation de vie.
- Résidence principale : "Vivez au rythme des Alpes toute l'année"
- Résidence secondaire : "Votre refuge familial à 6h de Paris"
- Investissement : "L'immobilier de luxe alpin : une valeur refuge"

## Application rapide par type de post

| Objectif | Levier principal | Exemple |
|---------|-----------------|---------|
| Propriété vedette | Rareté + Ancrage | "Unique à Megève. Vue frontale Mont-Blanc." |
| Post lifestyle | JTBD + Sensoriel | "Vos matins méritent cette vue." |
| Post marché | Autorité + Données | "Prix m² Mont d'Arbois : +8% en 2025" |
| Story CTA | Urgence douce | "Visite disponible ce week-end — DM" |
| Témoignage | Preuve sociale | Citation client + résultat concret |

## Ce qu'il faut éviter

- **Fausse urgence** : "Dépêchez-vous !" — destructeur de crédibilité luxe
- **Superlatifs non prouvés** : "Le plus beau chalet de Megève"
- **Prix sans contexte** : Un prix seul crée de la résistance, pas du désir
- **Trop de preuves sociales** : 1 par post maximum
