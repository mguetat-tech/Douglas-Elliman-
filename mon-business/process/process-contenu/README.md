# SOP — Création contenu Instagram

## Calendrier éditorial

| Jour | Type | Format |
|------|------|--------|
| Lundi | Bien du moment (anonymisé) | Carrousel 5 slides |
| Mercredi | Coulisses / lifestyle | Reels 15-30s |
| Vendredi | Éducation marché | Post unique + Story |

## Template carrousel "Bien du moment"

**Slide 1 — Accroche**
```
🏔️ [VILLE] — [ADJECTIF EXCLUSIF]
[1 phrase mystère sans révéler le bien]
↳ DM pour les détails
```

**Slide 2–4 — Photos intérieur**
- Grand angle salon principal
- Vue terrasse / ski room / piscine
- Détail signature (cheminée, cave, vue...)

**Slide 5 — CTA**
```
Vous cherchez [TYPE] à [VILLE] ?
Notre portefeuille off-market vous attend.
📩 DM ou lien en bio
```
