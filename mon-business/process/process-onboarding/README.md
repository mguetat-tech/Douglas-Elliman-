# SOP — Onboarding nouveau client

## Process vendeur (mandat)

### Étape 1 — Premier contact (J0)
- [ ] Appel découverte 20 min
- [ ] Remplir fiche dans `memory/project_deals.md`
- [ ] Envoyer email de confirmation + proposition RDV

### Étape 2 — Visite du bien (J3–J7)
- [ ] Préparer dossier estimation avec comparables
- [ ] Photos préliminaires iPhone
- [ ] Présenter services Douglas Elliman et réseau international

### Étape 3 — Signature mandat (J7–J14)
- [ ] Envoyer mandat exclusif par DocuSign
- [ ] Briefing photographe pro
- [ ] Créer fiche bien confidentielle

### Étape 4 — Mise en marché (J14+)
- [ ] Diffusion portails (Belles Demeures, partenaires Sotheby's)
- [ ] Créer série posts Instagram (voir `process-contenu/`)
- [ ] Alerter réseau acquéreurs via `tools/email-sender.py`

## Process acquéreur

### Étape 1 — Qualification (J0)
- [ ] Appel 30 min : budget, critères, timeline, financement
- [ ] Ajouter dans `memory/project_deals.md`
- [ ] Vérifier solvabilité

### Étape 2 — Envoi sélection (J1)
- [ ] 3–5 biens maximum
- [ ] Email personnalisé avec mise en contexte

### Étape 3 — Suivi hebdomadaire
- [ ] Check-in chaque lundi
- [ ] Adapter les critères après 3 visites sans match
